# Universal Interactive Error Page

A **fun, interactive, and fully customizable error page** that works for any type of error: 404, 500, or custom errors.  
Includes a **Go Back button**, **random confetti**, and **20+ hidden easter eggs** for playful user interaction.

---

## Features

- ✅ Universal usage: Works for any HTTP error.
- ✅ Minimalist design with a **Go Back button**.
- ✅ Random confetti and floating emojis.
- ✅ Multiple hidden easter eggs triggered by clicks, key presses, or console.
- ✅ Fully customizable colors, text, and animations.

### Included Easter Eggs

1. Random confetti on page load

4. Console easter egg: `oops()`  
5. Paragraph hover changes background

1. **Save the file** as `error.html` in your project.
2. **Link it** to your server’s error pages:

   - **Apache (.htaccess):**
   ```apache
   ErrorDocument 404 /error.html
